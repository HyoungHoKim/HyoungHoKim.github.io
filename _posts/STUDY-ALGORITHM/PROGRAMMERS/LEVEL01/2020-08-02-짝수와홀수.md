프로그래머스 코딩테스트 연습문제입니다. 딱히 설명이 필요없는 문제입니다. %연산자를 사용하면 됩니다.

### 문제 설명
정수 num이 짝수일 경우 Even을 반환하고 홀수인 경우 Odd를 반환하는 함수, solution을 완성해주세요.

### 제한 조건
- num은 int 범위의 정수입니다.
- 0은 짝수입니다.



### 풀이

```
#include <string>
#include <vector>

using namespace std;

string solution(int num) {
    string answer = "";

    if(num%2 == 0) answer = "Even";
    else answer = "Odd";

    return answer;
}
```
